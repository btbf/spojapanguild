# **エアギャップマシンの作成**

!!! summary "エアギャップマシンとは？"
    プール運営で使用するウォレットやプール運営の秘密鍵を<font color=red>オフライン</font>上で発行・管理し、秘密鍵を使用するトランザクションファイル署名作業等に使用します。

## 環境構築方法

VirtualBoxでの環境構築方法を2パターン用意しております。  

| セットアップ方法 |  特徴 | 作業内容 |
| ------------- | --------- | --------- |
| [自動セットアップ](./vb/air-gap-auto.md) | 簡単 | OVAプレビルドパッケージ 自動セットアップ |
| [手動セットアップ](./vb/air-gap-manual.md) | マニュアル必須 | VirtualBOX設定/Ubuntuインストール/SJG環境  手動セットアップ |

※お持ちのパソコンにUbuntuOSをネイティブインストールする方法も問題ございませんが、インストール方法は割愛させていただきます。
